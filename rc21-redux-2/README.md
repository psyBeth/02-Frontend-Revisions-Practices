# Redux Toolkit Example

## `Installation`

```
npm install or yarn
```

## `Libraries used`

- `@reduxjs/toolkit`
- `react-redux`
- `axios`
- `react-router-dom`
- `@mui/material-ui`
- `@emotion/react`
- `@emotion/styled`
